Tentang Saya

Saya adalah seorang pengembang web independen yang saat ini sedang membangun ZHStore, sebuah platform toko online berbasis afiliasi TikTok Shop yang dikembangkan sepenuhnya menggunakan PHP dan SQLite. Meskipun hanya berbekal perangkat mobile, saya dengan tekun mengembangkan sistem parsing otomatis untuk produk afiliasi TikTok, lengkap dengan metadata, gambar, harga, dan fitur update data per entri.

Proyek ini menjadi pondasi utama saya untuk membangun portofolio nyata di dunia pengembangan web. Saya juga pernah mengembangkan aplikasi sholawat serta situs volkleather.zhwifi.xyz (sudah tidak aktif), namun saat ini fokus penuh diarahkan pada pengembangan ZHStore.
